# Problem 3

입력 X가 있어서 X = 0이면 000 → 001 → 011 → 010 → 110 → 111 → 101 → 100 → 000 → ...으로 변하고, X = 1이면 000 → 001 → 010 → 011 → 100 → 101 → 110 → 111 → 000 → ...으로 변하는 카운터를 설계하고자 한다.

1. 상태전이도(state diagram)을 그리시오.
2. 1을 JK flip-flop으로 설계해 보시오.
